# AI Service Aggregator

This is a Node.js + Express + MongoDB based AI too aggregator platform，users can register and login, browsing AI toolss, comments and save etc.

## Folder structure

- backend/: backend scripts
- frontend/public/: frontend static scripts (JS/CSS)

## running env

- Node.js v18+
- MongoDB (local)

## how to use:

1. clone repo and change into directory of 'backend'：

   # bash
   cd backend

2. run below commands sequentially:

   npm install
   npm run seed
   npm start
